Installation¶

Dependencies¶

Already distributed with Anaconda and Canopy.

  • NumPy >= 1.6.1
  • SciPy >= 0.14
  • Matplotlib >= 1.5

You can also install using pip.

pip install numpy
pip install scipy
pip install matplotlib

Spykes¶

Clone the repository.

$ git clone http://github.com/KordingLab/spykes

Install spykes using pip as follows

$ cd spykes
$ pip install -e ./
