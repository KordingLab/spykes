Libraries¶

  • Fork the repository
  • Make changes on your local branch
  • Submit pull requests to be reviewed

Documentation¶

The requirements to build the documentation are

  • sphinx
  • numpydoc

Installation: pip install sphinx numpydoc

To build the documentation locally, run:

make html

Before you submit any documentation to the repo, clear any built files from your local repo using:

make clean
